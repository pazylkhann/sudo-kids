# SudoKids

**Образовательная платформа судоку для детей 6–12 лет с AI-наставником, удержанием через ежедневные челленджи и подпиской Pro.**

Демо: [sudo-kids.lovable.app](https://sudo-kids.lovable.app)

---

## Суть продукта

SudoKids — веб-приложение, которое учит логическому мышлению через судоку. Ребёнок играет на адаптированном поле (4×4, 6×6, 9×9), получает объяснения ходов от AI Coach, участвует в общем пазле дня и соревнуется в рейтингах. Родитель видит прогресс через профиль, стрики и достижения.

Продукт закрывает разрыв между массовыми судоку-сервисами для взрослых и потребностью родителей в осмысленной цифровой активности без бесконечного скролла.

---

## Проблема

| Сторона | Боль |
|--------|------|
| **Родитель** | Сложно найти приложение, которое одновременно развивает мышление, понятно ребёнку и не требует постоянного контроля контента |
| **Ребёнок** | Классические судоку перегружены интерфейсом, не объясняют логику хода и быстро теряют интерес |
| **Рынок** | Существующие решения ориентированы на взрослых; детский сегмент с персонализированным обучением и монетизацией подписки почти не занят |

---

## Решение

SudoKids сочетает полноценный игровой движок, педагогический слой (уроки техник) и AI-наставника, который отвечает в контексте текущей позиции простым языком. Механики удержания — daily challenge, стрики, стикеры, лидерборды (глобальный и по городу) — формируют регулярную практику, аналогичную модели habit-building в EdTech.

---

## Целевая аудитория

- **Основная:** дети 6–12 лет  
- **Платящая / принимающая решение:** родители  
- **Сценарий использования:** короткие ежедневные сессии (10–15 минут) как альтернатива пассивному потреблению контента

---

## Конкурентное преимущество

1. **Фокус на возраст** — UI, сложность и контент под ребёнка, а не адаптация взрослого продукта  
2. **AI Coach** — объяснение ходов и техник в диалоге, а не статичные подсказки  
3. **Социальный слой** — daily puzzle с общим лидербордом и соревнование внутри города  
4. **Прогрессия** — обучение от простых техник к сложным, геймификация через стрики и достижения  
5. **Готовая техническая база** — рабочий MVP с auth, БД, server functions и rate-limit для Free/Pro

## Бизнес-модель

**Freemium + подписка Pro**

| Тариф | Возможности |
|-------|-------------|
| **Free** | Игра, daily, базовый лимит запросов к AI Coach |
| **Pro ($4.99/мес)** | Расширенный лимит AI, кастомные темы, эксклюзивные стикеры |

Монетизация масштабируется за счёт конверсии активных пользователей daily challenge в подписчиков. Следующий шаг — подключение Stripe (webhook, автоматическая активация Pro).

## Технологии

| Слой | Стек |
|------|------|
| Frontend | React 19, TanStack Start (SSR), Tailwind v4 |
| Backend | TanStack server functions, Supabase (PostgreSQL, Auth, RLS) |
| AI | Lovable AI Gateway → Gemini 2.5 Flash |
| Инфраструктура | Lovable Cloud, миграции Supabase |

Архитектура разделяет клиент, серверные функции и БД; доступ к данным ограничен Row Level Security. AI-запросы лимитируются на уровне `ai_coach_usage` для контроля unit-экономики Free-тарифа.

**Ключевые модули:** `src/lib/sudoku/engine.ts` (генерация/решение), `puzzles.functions.ts`, `coach.functions.ts`, `profile.functions.ts`, маршруты `play`, `daily`, `leaderboard`, `profile`, `learn`, `pricing`, `auth`.

**Схема БД:** `profiles`, `puzzles`, `daily_puzzles`, `daily_results`, `game_sessions`, `achievements`, `ai_coach_usage`, роли через `user_roles` / `has_role()`.
